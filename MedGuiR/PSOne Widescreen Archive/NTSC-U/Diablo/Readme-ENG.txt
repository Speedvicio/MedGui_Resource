Diablo (PSX) - Extra Widescreen Hack (v1.0)
===========================================

About EWS Hack:
~~~~~~~~~~~~~~~
- This hack provides support for extra widescreen image (32:15) without distortion of any sprites proportions (HUD, characters, special effects, etc.).
- Warning!!! This hack does not work correctly on the original hardware. Use PS1 emulators for various platforms.
- To apply this hack you need the original USA version of the game (SLUS-00619) and "PPF O MATIC" tool.

	MD5 (data track):	f252563de50dbdf0d54dd6da5e8cb057
	CRC32 (total):		bdfccdc3


Changes in version 1.0:
~~~~~~~~~~~~~~~~~~~~~~~
- Support for extra widescreen image (32:15).
- Centering of splash screens, menus, messages and others.
